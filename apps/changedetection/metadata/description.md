## ChangeDetection.io with Playwright

Self-hosted website change detection and notification service.

### Features
- Monitor any website for changes
- Playwright browser support for JavaScript-rendered pages
- Telegram, Discord, Slack, email notifications via Apprise
- Visual selector for choosing what to monitor
- API for automation
- CSS/XPath/JSONPath filters

### Playwright Support
This version includes sockpuppetbrowser for rendering JavaScript-heavy pages. After installation, go to **Settings → Fetching** and select **"Playwright Chromium/Javascript"** as the default fetch method.
