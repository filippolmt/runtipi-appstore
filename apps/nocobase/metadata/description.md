NocoBase is the most extensible open-source no-code/low-code platform for building business applications and enterprise solutions.

## Features

- **Data Model Driven**: Define your data models and NocoBase automatically generates the UI
- **Plugin Architecture**: Highly extensible through plugins - add features without modifying core code
- **Workflow Automation**: Built-in workflow engine for automating business processes
- **Role-Based Access Control**: Fine-grained permissions and access control
- **API First**: Every feature is accessible through RESTful APIs
- **Multiple Data Sources**: Connect to PostgreSQL, MySQL, MariaDB, and more
- **Rich UI Components**: Forms, tables, charts, calendars, kanban boards, and more
- **File Management**: Upload, manage, and organize files and attachments
- **Audit Logs**: Track all changes and actions in the system
- **Multi-language Support**: Internationalization built-in

## Use Cases

- Internal tools and admin panels
- CRM and project management
- Inventory and order management
- Content management systems
- Custom business applications
