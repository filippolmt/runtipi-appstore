# Workout Cool

Modern open-source fitness coaching platform that enables you to create workout plans, track progress, and access a comprehensive exercise database.

## Features

- **Workout Plans**: Create and customize your own workout routines
- **Progress Tracking**: Monitor your fitness journey over time
- **Exercise Database**: Access a comprehensive library of exercises with detailed instructions and video demonstrations
- **Responsive Design**: Works seamlessly on desktop and mobile devices

## Getting Started

After installation, access the app through your browser. On first launch, you can optionally seed the database with sample exercises to get started quickly.
